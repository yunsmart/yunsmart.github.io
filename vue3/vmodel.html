# VModel

注意

  • 用于自定义组件时，v-model prop 和事件默认名称已更改：

    • prop: value -> modelValue
    • event: input -> update:modelValue

  • v-bind 的 .sync 修饰符和 组件的 model 选项已移除，统一为 v-model参数形式。

  • 可以自定义 v-model 修饰符

  • 示例：

    • 默认写法
    <!-- ModelComp.vue -->
<template>
	<div @click="$emit('update:modelValue', modelValue + 1)">
		{{ modelValue }}
	</div>
</template>

<script>
	export default {
		props: {
			modelValue: {
				type: Number,
				default: 0,
			},
		},
	}
</script>

	<!-- 相当于 -->
	<model-comp :modelValue="count" @update:modelValue="count = $event" />
</template>

<script>
	import { ref } from 'vue'
	import ModelComp from './ModelComp.vue'
	export default {
		setup() {
			const count = ref(1)
			return { count }
		},
	}
</script>
